# Платформер игра

## Описание проекта

Это веб-игра в жанре платформер, реализованная на TypeScript с использованием современных технологий фронтенд-разработки. Игра использует Canvas для отрисовки графики и WebSocket для взаимодействия с сервером в реальном времени. Проект разработан с использованием Vite как сборщика модулей и TypeScript для обеспечения строгой типизации.

## Архитектура проекта

```
src/
├── entity/                 # Классы игровых объектов
│   └── hero/               # Классы, связанные с героем
│       ├── Hero.ts         # Основной класс героя
│       └── hero-animations.ts # Анимации героя
├── game/                   # Основная логика игры
│   ├── AssetsFactory.ts    # Загрузка и управление ресурсами
│   ├── Camera.ts           # Класс камеры для отслеживания игрока
│   ├── Game.ts             # Основной игровой цикл
│   ├── HeroStore.ts        # Хранилище данных героя
│   ├── KeyboardSendMessage.ts # Обработка клавиатурных событий и отправка сообщений
│   ├── StaticBackground.ts # Статический фон
│   ├── WebSocketGame.ts    # Подключение к веб-сокету
│   ├── World.ts            # Мир игры
│   └── WsMessageProcessor.ts # Обработка сообщений веб-сокета
├── shared/                 # Общие утилиты
│   └── keyboard/           # Обработка клавиатуры
│       └── KeyboardProcessor.ts
├── types/                  # Типы TypeScript
│   ├── Assets.ts           # Типы для ресурсов
│   └── KeyBoardTypes.ts    # Типы для клавиатурных событий
├── main.ts                 # Входная точка приложения
└── vite-env.d.ts           # Определения типов для Vite
```

## Зависимости и настройка окружения

### Зависимости

Проект использует следующие основные зависимости:

- TypeScript (^5.2.2) - язык программирования с поддержкой типизации
- Vite (^5.2.0) - быстрый сборщик модулей для современных веб-проектов

### Установка и настройка

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Запустите в режиме разработки: `npm run dev`

## Запуск и использование

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173

### Сборка проекта

```bash
npm run build
```

### Просмотр собранного проекта

```bash
npm run preview
```

## Особенности игры

- Игровой мир размером 5000x768 пикселей
- Камера, следящая за игроком (ширина 1024 пикселя)
- Поддержка полноэкранного режима по двойному клику
- Взаимодействие с сервером через WebSocket
- Клавиатурный ввод для управления персонажем
- Система анимаций для персонажа
- Система загрузки и управления игровыми ресурсами

## Технологии

- TypeScript - для строгой типизации и лучшей поддержки кода
- Canvas API - для отрисовки графики
- WebSocket - для связи с сервером в реальном времени
- Vite - для быстрой сборки и разработки
- ООП подход - для организации кода

## Структура игрового процесса

1. Загрузка ресурсов через AssetsFactory
2. Создание мира игры (World)
3. Инициализация камеры (Camera)
4. Создание хранилища данных героя (HeroStore)
5. Подключение к WebSocket серверу
6. Обработка клавиатурного ввода
7. Запуск игрового цикла с анимацией

## Управление

- Клавиши стрелок - движение героя
- Двойной клик - переключение полноэкранного режима
